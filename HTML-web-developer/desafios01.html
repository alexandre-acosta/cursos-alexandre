Aprendendo HTML na Prática

Este site é um exercício para aplicar tags fundamentais.

1. Formatação de Texto

Subtítulo de Nível 3

Podemos escrever de forma negritada (strong), itálica (i) ou até mesmo sublinhada (u). Às vezes, queremos destacar algo com marca-texto (mark) ou escrever um texto bem pequeno (small).

Exemplos Matemáticos e Químicos

A fórmula da água é H2O (sub) e o Teorema de Pitágoras é a2 + b2 = c2 (sup).

Tags Sugeridas (Pesquisa)

Aqui usei a tag texto riscado (del) para mostrar algo que foi removido. Também usei a tag HTML (abbr), que mostra o significado ao passar o mouse. Além disso, embora seja antiga, a tag mudei a cor e fonte aqui (font).
